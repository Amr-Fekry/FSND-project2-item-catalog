Method GET

/api/categories

      },

    Usage

    For a specific category, obtain category's name and a list of category items.


    Making a Request
    Method GET
    Base URL Path 
    /api/category/<category_id>
    Replace <category_id> with the equivalent id from /api/categories
    Parameters None

    Responses

  • Upon a successful request:

    • 
  {
    "category_id": <category id>,
    "category_name": <category name>,
    "category_items": [
      {
        "item_id": <item id>,
        "item_name": <item name>,
      },
      .
      .
    ]
  }
    Usage

    For a specific item, obtain item's name and description and category's id and name.


    Making a Request
    Method GET
    Base URL Path 
    /api/item/<item_id>
    Replace <item_id> with the equivalent id from /api/category/<category_id>
    Parameters None

    Responses

  • Upon a successful request:

    • 
  {
    "category_id": <category id>, 
    "category_name": <category name>, 
    "item_description": <item description>, 
    "item_id": <item id>, 
    "item_name": <item name>
  }
